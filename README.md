🔐 CipherVault — Secure Credential Storage System

CipherVault is a full-stack secure password and credential management system built with modern web technologies. It demonstrates real-world backend security engineering, encryption, authentication systems, and scalable architecture design.

🚀 Features
🔑 Authentication & Security
JWT-based authentication (Access + Refresh tokens)
HTTP-only secure cookies
Password hashing with bcrypt
Two-Factor Authentication (TOTP)
Rate limiting protection
Role-based access control (RBAC)
🏦 Vault System
Create multiple secure vaults
Organize credentials by category
Search & pagination support
Ownership-based access control
🔐 Credential Manager
AES-256 encrypted password storage
Secure credential CRUD operations
Decryption only at runtime
Website, username, notes support
📊 Security & Logging
Audit logging for all actions
Activity tracking system
Login/logout tracking
Secure session handling
💻 Frontend
React + TypeScript UI
Tailwind CSS styling
Protected routes
Dashboard layout
Password strength meter
Password generator
⚙️ DevOps & Testing
Docker containerization
Docker Compose setup
GitHub Actions CI pipeline
Unit testing with Jest
Supertest API testing
